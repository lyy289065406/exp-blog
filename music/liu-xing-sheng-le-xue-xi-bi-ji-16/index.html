「流行声乐学习笔记 16」混声体系和换声概念

0x10 【直播】随堂流水笔记

0x11 音域与音色

  • 大提琴是可以拉高音的（所有乐器都能发出高音）、但是小提琴不能拉低音（琴弦决定了）
  • 低音音域是声带先天条件决定的；而高音音域是可以后天锻炼的，所有人经过练习都能唱高音
  • 小孩的声带天生都是薄短的，只有男生的声带会二次发育变得厚长，这也是为什么女生普遍更容易唱高音，因为她们的声带和小孩是几乎一样的
  • 天生： 罗西尼男高音 —— 国内唯一周深
  • 后天培养： 16 世纪 ~ 18 世纪欧洲存在阉人歌手，为了不让声带二次发育就阉掉，也叫阉伶、去势男高音
  • 声带薄短的人会更容易悟出环甲主导的发声习惯（高音轻松）；声带厚长的人会更容易悟出甲杓主导的发声习惯（低音轻松）
  • 声带厚长的人需要学习使用环甲拉薄声带，以支持唱高音
  • 声带越长的人上限越高（音域越宽）、在发声技术相同的前提下，其天赋比声带短的人更好
  • 通过甲杓肌可以改变声带与空气的接触面积、控制声带的厚/薄进行发声
  • 环甲控制拉长声带（变薄、唱高音）、甲杓控制声带接触面积（变厚、唱低音）
  • 天生声带厚的人需要经常做环甲拉伸动作，使得声带变薄

0x12 混声体系

  • 期中考之前是把气息、发声、声带控制等单独拎出来学，期中之后就要开始学搭配了
  • 练习正确发声的目的是把这些技巧变成被动技能
  • 音乐体系：
    • SLS（Speech Level Singing）：由 Seth Riggs 创立（已解散），让歌唱时声带保持自然平衡，如同说话一样。
    • EVT（Estill Voice Training）：由 Jo Estill 创立，通过对声道各构音结构的独立控制，实现多样化音色与技巧。
  • 混声不算是技巧，它是搭配的一种声音
  • 说话状态就是一个最典型的混声状态，只不过一般人不经过训练，无法把说话状态保持到高音区（喉咙会开始改变发力、或用到了假声，导致状态突变了）
  • 强混就是气流和肌肉会更多
  • 三种混声（弱混、强混、平衡混）的区别其实只是音量的区别
  • 只要音高上去了、就一定有声带拉长

0x13 换声点与换声区

  • 参考混声的概念，其实没有所谓的换声点，换声点是个假名词，换声点不过是肌肉没控制好导致的破音而已
  • 喉舌分离体感不舒服是正常的，没有人在改变习惯的时候还会觉得很舒服

0x20 【录播】重点纪要

0x21 概念



  • 声带越短的人音域越窄（低音下不去、但高音音域相对较宽），类比琴弦越短就只能发出振动频率越高的声音
  • 声带越长的人音域越宽（整体音域较宽），类比琴弦越长就只能发出振动频率越低的声音
  • 声带越薄的人音色越细（高音轻松）
  • 声带越厚的人音色越粗（高音需要声带拉长支持）
声带条件 音域 音色 说明
短 + 薄 低音音区有限、高音相对轻松 较细 天生具备唱高音条件
短 + 厚 低音音区有限、高音相对困难（需要技术支持） 较粗 经过练习可以高音
长 + 薄 音域较宽、所有音区相对轻松 较细 天生就是歌手
长 + 厚 音域较宽、高音相对困难（需要技术支持） 较粗 经过练习可以唱全音域的歌

每个人声带天生不一样，模仿别人会带来负面影响，除非声带条件特别相似

0405-01:08:29 或 0411-01:00:56 如何测试自己声带长短：

  • 通常情况下，一个八度的 1234567 对应的音名为 CDEFGAB
  • 在钢琴中把键盘分成若干组，每组对应八度音高
  • 通常我们唱的 1234567 音高在第 4 组，即标记为 C4、D4、E4、F4、G4、A4、B4
  • 第 2 组往前称为低音区（Low）
  • 测试声带长短时，从 C3 开始往前倒音阶：
    • 如果只能唱 Low-B、Low-A，再往前 Low-G 开始出现气泡音了，说明就是短声带
    • 如果能唱到 Low-G ~ Low-C 甚至更靠前的音，说明就是长声带

长厚声带的同学可以多听 王力宏 的演唱技巧，早期的（2001年） 王力宏 唱歌声带长厚的特点非常明显，后面每年一次的福利秀发声技术就越来越好

0405-01:13:25 如何测试自己声带厚薄：

  • 参考平时说话，放松情况下：
    • 声音靠前、扁尖，就是薄的
    • 声音靠后、浑浊，就是厚的
  • 一般人都是处于厚薄适中、偏哪方便多一些而已

我是 偏长 + 偏厚（可以发 Low-C），有不错的先天条件，主要练习环甲拉长使得声带变薄唱高音


  • 混声：发声时发力统一的气流与肌肉的状态（状态不突变），它不是一种 “音色” 或 “声音”
  • 混声的前提： 真声为前提、不能发假声，假声是没有任何肌肉参与的声音、只有空气流动，全程挡不住气使不上劲（假声可以作为感受拉长的动作、或者改变音色，但是假声绝对不在混声的讨论范围内）

混声是改变音高的过程中：统一的、不断层的、没有换声点的声音 —— 在这个状态下发出的所有声音（除了假声）都可以称为混声

唱歌到底要不要用力？

0405-01:38:16 举个例子解释就明白了：

  • 把手臂平放悬浮在胸前，保持住
  • 你事实上用力了，不然手臂不会悬浮；但你感觉不到这个力，因为你的手臂没有发抖（但维持越久你会越明显感觉到自己在用力）
  • 这种就是大小适中的力、为了保持状态而自动出现的力
  • 唱高音其实是一个道理，唱高音要拉长环甲、就必须要发力拉长到合适位置，可以理解为是一种暗劲
  • 高音的两种极端是：
    • 完全没肌肉参与的假声，完全没用力
    • 用力过猛导致的挤嗓，甲杓主导的错误的力
  • 环甲维持发力，就是要介于这两者之间，甚至在养成被动习惯后，就感觉不到环甲在发力（和感觉不到手臂悬浮发力一个道理）
  • 某些声乐老师可能会说高音不能发力，其实是相对挤嗓状态而言，好像没在用力，但不发力是无法正确唱高音的，实际上就是会用到维持高音区说话状态的力、刚好够用就可以了，不是挤压的力
  • 唱歌喉咙（环甲）要发力，而且音越高则越发力
  • 在中低音区说话都需要用一点点维持说话状态的力度，音高越高自然就需要更多维持说话状态的力、因为会越来越难做
  • 音高越高、音调越高、空气流速越快，此时如果还要维持说话，用来对抗的力就需要更多

理解即可，非必要可不用

  • 不存在换声点，换声点不过是肌肉没控制好导致的破音而已
  • 参考第 05 节真假声切换，就可以做到平滑没有换声点

  • 轻机能 = 弱混 = 头混 = 2档
  • 重机能 = 强混 = 胸混 = 1档
  • 面罩 = 环甲肌主导的声音
  • 关闭 = 声带闭合
  • 喉舌分离 = 发声肌肉 和 咬字肌肉 无关
  • 弱混、强混、平衡混 之间的差别在于音量的大小（给人的感觉就是上下起伏的差异，在音色统一的情况下这种海浪感就会让人很舒服）
  • 金属芯： 只有在闭合良好的状态下声音才有的金属小小泛音亮亮的听感（振动频率极高时隐藏在声音里面滋滋很尖的电流声，环甲很难找到这个声音、但气泡音可以找到）

做混声的时候不要想着怎么做强，要想着怎么把声音放出去（推气、整体的发力感）

0x22 正确的行为

届 进度条 行为描述
0405 01:23:26 在喉咙拉长/缩短声带（改变音高）的时候，能够始终保持住声音、气息支撑、肌肉发力、统一不突变，就是做到了混声（环甲和甲杓一直在工作、发力没有忽大忽小）
0405 01:35:50 要想把说话（混声）状态保持到高音区，一定是喉咙（环甲）不断层地越来越用力的（注意不是甲杓用力/挤嗓，正确用力只有环甲）
0411 01:17:12 发声统一不突变：声带在喉咙力拉长/缩短的时候，即环甲和甲杓在工作并且改变音高的同时，能够始终保持支撑的肌肉发力统一不突变。简而言之，就是让环甲和甲杓一直工作到最高的音、然后再回来到最低那个音、也依旧在工作，每个音都在工作、而且发力感是统一的、不是忽大忽小一会工作一会不工作，这个状态下发出的所有声音就是混声
0411 01:22:46 之前课程一直说环甲主导、环甲要工作，从混声开始甲杓也要参与工作，但是还是环甲主导（环甲比例更多）
0411 01:29:11 用环甲在高音区说话： 模仿并保持幼儿园女老师说话、轻薄+轻柔+清晰，保持住到高音区又能变成说话了（不要变成假声）。这件事情需要发力，因为想把说话状态保持到高音区，它一定时喉咙越来越用力，但不是发那种牛劲挤上去地力，而是为了保持说话力道发高音而自动出现的力（类比把胳膊维持悬浮在半空的发力状态）

0x23 错误的行为

届 进度条 行为描述
0405 01:21:46 不要拿假声去唱歌，假声是全程漏气的，它可以练习声带拉长本身，但是不能用于唱歌
0411 01:19:37 没有人会故意很虚或者超级用力去说话，所以平常说话就是典型的混声，因为肌肉发力不会突变，但是没受过训练的人很难把这种状态保持到高音区、一往高音走就会开始改变喉咙的发力状态（松开了声带、漏气、断层、越高越挤、用力过猛崩溃等）
0411 01:22:47 声带又长又厚的男生很容易变成甲杓主导的情况，虽然知道环甲和甲杓要同时工作、也知道环甲要主导，但因为习惯了低音区的环境，所以保持状态的时候音高会一直憋在喉咙里，纠正的方式是去辨识环甲、说话时要找环甲、用环甲去讲话（例如模仿幼儿园的女老师轻声细语又清晰地说话）
