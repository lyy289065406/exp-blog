Switch 使用 DBI 安装免费游戏笔记

0x00 前言

书接前文《Switch 大气层与系统版本升级笔记》，升级系统的目的肯定是为了玩最新的游戏。

但毕竟 Switch 压箱底吃灰太久，几乎都忘记如何安装游戏了，所以本文特意把过程记录下来。

0x10 升级前准备

对于大气层 Switch 系统而言，DBI 就是安装 NSP/NSZ/XCI/XCZ 格式游戏的终极工具。

它提供了 MTP 技术用以高速便捷的安装/卸载游戏，其入口在：

  1. Switch 进入虚拟系统
  2. 使用数据线直连 Switch 主机和电脑
  3. 依次点击【相册】【DBI】

如果【相册】里面找不到 DBI，则需要先安装它：

  • 在官方 Github 发布页面下载 dbi.config 和 DBI.nro
  • 把 dbi.config 和 DBI.nro 复制到 SD 卡的 SD:/switch/DBI/ 目录下即安装完成

0x20 下载免费游戏

免费的 swicth 游戏比较喜欢去这两个渠道搜索下载：

  • swicth520: https://www.gamer520.com/
  • 520switch: https://www.520switch.com/

这两个渠道我也说不上谁抄谁，反正免费就行。

譬如近期最热门的大 IP 游戏都可以在里面找到：

  • 《宝可梦传说Z-A》
  • 《空洞骑士 丝之歌》

通常下载游戏后，可能会有以下安装包：

  1. 游戏本体
  2. 游戏补丁
  3. 游戏 DLC

只要安装包后缀是 NSP/NSZ/XCI/XCZ 格式之中的，都能够安装。

0x30 安装游戏

主要通过 DBI 操作，步骤如下：

  1. Switch 进入虚拟系统
  2. 使用数据线直连 Switch 主机和电脑
  3. 依次点击【相册】【DBI】【运行 MTP 响应器】

不同版本的 DBI 菜单可能不一样，总是记得 MTP 关键字的菜单点进去就好

进入 MTP 后电脑上会多挂载了一个 Switch 的硬盘：

双击打开 Switch 硬盘后，可以看到里面有多个盘符：

  • 1: SD Card
  • 5: SD Card install
  • 6: NAND install
  • 7: Saves
  • 8: Album
  • 金手指和MOD放这
  • 相册工具

而用来安装游戏的，是 5: SD Card install。

把前面下载的游戏，脱拽进 5: SD Card install 即可自动解包安装：

大多情况下，安装顺序都是：本体 -> 补丁 -> DLC

在 Switch 的 DPI 可以看到更详细的安装过程：

安装完成后，在首页就能看到游戏了：

打开游戏可正常游玩即可：

0x40 卸载游戏

如果 SD 卡存储空间不够，可以卸载一些不常玩的游戏释放空间。

也是通过 DBI 操作，步骤如下：

  1. Switch 进入虚拟系统
  2. 使用数据线直连 Switch 主机和电脑
  3. 依次点击【相册】【DBI】【浏览已安装的应用程序】

通过上下移动光标，选择希望卸载的游戏，按 + 弹出操作菜单，光标移到【删除】上面按 A：

有些名字看不懂没关系，右边会展示游戏封面图，很好辨认。

根据个人需要选择是否保留存档，光标移到对应选项按 A：

确认准备删除的游戏信息，没问题则把光标移到【是的，我确认】按 A：

等待卸载完成后，按 B 返回即可：

0x50 后记

这里我再种草一个跨平台的全能模拟器 RetroArch：

  • 官网：https://retroarch.com/?page=platforms
  • Github： https://github.com/libretro/RetroArch

它支持安装在超多系统：Windows, macOS, Linux, Switch 等等 …

而且集成了超多主机模拟器：GB, GBA, PS2, PS3, PSP, PS Vita, Wii, Wii U, 2DS, 3DS, Switch 等等 …

只需要在 Switch 安装 RetroArch，就能畅玩其他主机平台独占的童年大作啦：

详见《Switch RetroArch 万能模拟器安装使用笔记》
