0x00 前言

在 RO 客户端中，窗口右侧会显示各种 Buff / Debuff / 特殊状态的图标：

其实增加、修改这些状态的图档和说明并不困难。

0x10 客户端配置

状态图档的配置文件都在目录 data\luafiles514\lua files\stateicon 下：

配置文件 说明 配置样例
efstids.lua 状态代码 = EFST ID EFST_PROVOKE = 0 ,
stateiconinfo.lua 状态文字说明 StateIconList[EFST_IDs.EFST_PROVOKE ] = { haveTimeLimit = 1, posTimeLimitStr = 2, descript = { {"挑衅", COLOR_TITLE_BUFF}, {"%s", COLOR_TIME}, {"攻击力上升，防御下降"} } }
stateiconimginfo.lua 状态代码 = *.tga 图档 [EFST_IDs. EFST_PROVOKE ] = " EFST_0000.tga ",

注意：

  • 在 efstids.lua 的 EFST_IDs 定义的 ID 没有连号的要求
  • 在 efstids.lua 的 EFST_IDs 定义的 ID 需要与服务端 src/map/status.hpp 枚举值一致
  • 在 efstids.lua 的 EFST_IDs 定义的 ID，其对应的状态代码必须在 stateiconinfo.lua 的 StateIconList 中使用，否则从断号开始、后面的状态都无法在客户端右侧栏显示
