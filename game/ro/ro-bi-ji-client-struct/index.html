0x00 前言

本文主要为 RO 客户端的基础知识科普，介绍了客户端目录结构、中文/韩文路径对照、各个图档的作用等等（相当于字典），便于个性化定制客户端时查阅、以快速找到需要修改的地方。

其中本站（https://exp-blog.com）在 2024 主要修订了：

  1. 职业外观的目录说明
  2. 武器外观的职业目录说明（此前三转沿用二转的武器图档目录已经不适用了）
  3. 新增宠物进化档位置

0x10 档案类型

文件后缀 说明
act(Action) 动作文件，控制 spr 图片零件的动作，包含长宽、缩放、定位、颜色、透明、旋转、延迟等等
spr(Sprite) 静态的图片分割文件（图片零件），跟 act 成对（头饰、发型、角色、魔物、外观服装等等等）
bmp(BitMaP) Windows 位图档，最常用到的类型（物品图、小地图、面版、NPC 大图 illust 等等）
tga(Targa) 特效图片，地图特效等等（可渲染）
str effect 特效檔，技能特效（不可渲染）
pal(Palette) 染色档或调色档色票（发色、职业外观染色）
ebm(Emblem) 工会图快取
wav(Waveform) 音频文件
gat(Ground Altitude) 地图属性文件（地图的属性包含地图大小、可否行走、攻击限制等等）
gnd(Ground Map) 地图地型档（地型高度、也包含纹理贴图）
rsw(Resource World) 地图配置文件（地图所有模块、路径、特效、光源、模块放的位置高度等等）
rsm(Resource Model) 地图模块文件（每栋建筑、树、对象的 3D 模型）
ini(Initial) INI 参数档（设定档）
xml XML 格式档案（用在设定服务器信息、宠物对话、怪物对话等等）
lub lua 脚本设定档，RO 大部份设定都改成 lub，一般为加密状态，解密有时候会改为 lua
txt 文字文件，最基础的档案
jpg 部分图片可用 jpg 取代，例如 Loading 图
imf 这是一个新的格式，主要针对各脚色修正 act 外观的偏移
gr2 类似 rsm 的功能，granny 3D 的模块

0x30 data

需要把 data.grf 解包后才能得到 data 目录

文件/目录 说明
data/wav 游戏音效、技能、攻击、怪物声等
data/wav/effect 特效音效
data/model 地图的 3D 模块数据
data/book 这应该不用解释了，书的内容
ba_frostjoke.txt 诗人的冷笑话
buyingstoreitemlist.txt 可以在收购商店收购的物品编号
carditemnametable.txt 卡片别名显示在前辍（注：前后都不设定，预设显示在前）
cardpostfixnametable.txt 卡片别名显示在后辍（注：前后都设定，显示在后）
cardprefixnametable.txt 卡片插入装备后，装备上显示的别名
clientinfo.xml 服务器信息，改 IP 或是 GM 外观
dc_scream.txt 舞者的惊声尖叫
etcinfo.txt 地图特效（snow sakura pokjuk smog fog）
exceptionminimapnametable.txt 新版功能，不显示地图放大的地图
fogparametertable.txt 雾化效果设定
GuildTip.txt 公会提示
indoorrswtable.txt 有室内视野限制的地图
itemmoveinfov5.txt 新版功能，物品右键内容显示的移动锁定
manner.txt 不雅对话限制
mapnametable.txt 地图名称
metalprocessitemlist.txt 技能制作物品的材料讯息
monstertalktable.xml 魔物对话
mp3nametable.txt 地图音乐 BGM 设定
msgstringtable.txt 客户端 MSG，所有的讯息都是在这翻译
num2cardillustnametable.txt 卡片左上（显示）卡片图鉴 档案位置 > cardbmp
pettalktable.xml 宠物对话
questid2display.txt 被动任务显示内容
resnametable.txt 地图映射
viewpointtable.txt 室外特殊视野限制

data 根目录下的 gat、gnd、rsw 都是地图档；其它档案可能是 旧版用的 或 没用的
