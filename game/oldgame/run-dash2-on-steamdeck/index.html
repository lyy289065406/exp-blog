0x00 前言

《洛克人Dash》是 CAPCOM 首款完全 3D 化的作品，融合动作、射击与角色扮演类型，由新一代的洛克人佛纳多担任主角。

而《洛克人Dash2：庞大的遗产》(Rockman Dash 2) 则是继《洛克人Dash：钢铁之心》后的续作、亦是 3D 洛克人系列登陆 PSP 主机第二作，Dash2 比 Dash1 的玩法更成熟、画质更细腻。

游戏叙述了一个在挖掘古文明遗迹的冒险年代，主角为了追寻传说的“庞大的遗产”而进行的冒险故事。游戏中有各式各样千奇百怪的支线任务静静等待玩家挑战，浓烈角色扮演与冒险解谜要素、以及迷你游戏杰出设计都是特色之一。游戏采用了好感度游戏系统，所有登场人物不再单纯扮演刻板角色，将会随着与洛克好感度高低产生不同效果。

可惜 Dash 系列没有官方汉化，只有 PC 版存在民间汉化。但是 Dash 的操作模式其实是更适合掌机的，如果要在 PSP 上玩，就只能玩日文版或者英文版。

有没有办法在掌机上玩到汉化版呢 ？ 答案是： SteamDeck 完美支持 !

0x10 准备工作

这是用 SteamDeck 原生系统运行复古老游戏的第二期！

因为《洛克人Dash2》不属于 steam 官方游戏，因此在安装 Dash2 之前，需要做一些准备工作：

  1. SteamDeck 切换到桌面模式：
    • 设置管理员密码
    • 启用 SSH 服务
    • 安装插件商城 Decky Loader
    • 安装兼容层管理插件 Wine Cellar
  2. PC 电脑安装 FTP 客户端 FileZilla
  3. 把 wifi 路由器分配给 PC 电脑和 SteamDeck 的 IP 分别固定（绑定 MAC）

这些准备工作可以参考第一期的《掌上仙境？教你如何在 SteamDeck 原生系统上运行 RO ！》，本文不再展开

0x22 上传 Dash2 到 SteamDeck

PC 版安装完成后，我们就可以上传到 SteamDeck 了。这里我们不要上传到 SteamDeck 的硬盘，而是安装在 TF 卡。

在 SteamDeck 中， TF 卡的位置是在 /run/media/deck 下，例如我的是 SD256 卡，那么这张卡的根目录就是在 /run/media/deck/SD256，于是游戏的安装目录就在： /run/media/deck/SD256/steamapps/common/。

现在可以通过 FileZilla 把整个 Dash2 目录上传到 /run/media/deck/SD256/steamapps/common/ 目录下：

0x30 配置：通过 Steam 启动 dash2.exe

切换到 SteamDeck 桌面模式，启动桌面上的 Steam :

  1. 在游戏库中，点击左下角的 添加游戏 -> 添加非 Steam 游戏
  2. 点击 浏览，切到刚刚上传 Dash2 的目录 /run/media/deck/SD256/steamapps/common/Dash2
  3. 选中 dash2.exe，点击 Open

此时 dash2.exe 会出现在游戏库列表中了。

修改封面图、专辑图等可以参考第一期的《掌上仙境？教你如何在 SteamDeck 原生系统上运行 RO ！》，本文不再展开

0x50 设置控制器按键布局

启动 Dash2 后，先设置控制器按键布局。

Dash2 本来是支持 PSP 手柄按键的，但是无法识别 SteamDeck 的手柄按键，因此只能基于键盘按键去修改：

基于此缘故，默认情况下 SteamDeck 最合适的控制器布局应该是 键盘（WSAD）和鼠标。

因此配合 Dash2 的默认键盘按键、去修改 SteamDeck 控制器的按键映射即可，下面是我根据个人习惯配置好的按键映射：

在这个按键布局的情况下，基本操作是：

  • L2: 鼠标右键 - 锁定 （比 PSP 同时按下 LR 方便）
  • R2: 鼠标左键 - 射击
  • L1: Q - 左转身
  • R1: E - 右转身
  • A: 空格 - 场景交互/对话交互/走路/特殊移动（需要搭配冲刺鞋）
  • B: Esc - 取消交互
  • X: L-S - 即左边的 Shift 键，跳跃
  • Y: L-C - 即左边的 Ctrl 键，特殊武器攻击
  • 左上角的两个方框 option： F1 - 游戏内菜单
  • 右上角的三横按键 command: 回车 - 确认/跳过过长动画
  • 左摇杆按下: F9 - 一次返回主菜单，两次退出游戏

同时要修改右摇杆 Y 轴反转，不然默认情况下上下转向与一般人的直觉是相反的：

嫌麻烦可以去社区找到我的配置下载，我已经分享了布局，名字为《Dash2 洛克人2庞大的遗产》：

按键布局根据自己需求随时调整即可，这里不再展开。但如果你修改了游戏内的默认按键，要注意不能使用 SteamDeck 的退出功能，必须要使用 F9 退出游戏才能正常保存设置，否则再次进入游戏时会恢复默认配置

0x60 设置屏幕拉伸

SteamDeck 的屏幕比 PSP 要大而且宽高比不一致，会导致 Dash2 屏幕左右有黑边。

如果希望满屏的话，可以在游戏中的 画面调整 中选择最大的 1280 x 800 x 32

与按键映射一样，修改屏幕分辨率后，必须要使用 F9 退出游戏才能正常保存设置，否则再次进入游戏时会恢复默认配置

0x70 Dash2，启动！

至此已经可以在 SteamDeck 上面玩 Dash2 咯 ~，点击 开始游戏 冲吧！
